# Procedure scorecard

Score 0–2 per dimension. Self-check before declaring done; also the scoring sheet
for the 3-arm benchmark. **Pass = every dimension ≥1 and the VERIFY gate satisfied
with cited evidence.** Any 0 means not done.

| Dimension | 0 | 1 | 2 |
|---|---|---|---|
| Goal reframing | echoes the literal ask | restates intent | intent + constraint envelope + non_goals |
| Context selection | topical / broad | role-justified | role-justified + precedent to mirror + hazards |
| Constraint provenance | asserted | some evidence links | every constraint tiered + evidence-linked |
| Alternative rejection | none / by taste | named reason | category + broken-boundary; path removed not guarded |
| Risk | effort-based or vague | severity + mitigation | blast-radius severity + runnable mitigation mirrored to acceptance |
| Acceptance | prose | some commands | all runnable; behavioral test + artifact check; falsifiable |
| Validation loop | no evidence graph | one-way refs | first real read revises plan; decisions cite observations |
| Failure handling | silent drop / fabricated | rejects/defers | fails closed; defers authority; honest carry-forward |

Total ≥14/16 with no zeros = full procedure exhibited. Reference traces in the
source corpus scored 14–16; un-worked skeletons scored 0–2.

> Scoring the gap, not the model: this measures whether the *process* was
> followed, which is the transferable part. It does not measure raw capability —
> that is what the host model brings and what the benchmark isolates.
